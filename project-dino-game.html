Mini jeu type Dino Game

Projet SAE IE en binôme : développement d'un mini jeu inspiré du Dino Game de Chrome, mêlant programmation, graphismes simples et gestion des collisions.

← Retour aux projets Code source

Rôles & tâches

  • Programmation du gameplay : déplacements, sauts et gestion des collisions.
  • Création des assets simples et intégration des sprites.
  • Gestion des scores et boucle de jeu fluide.

Résultats

  • Prototype jouable et stable.
  • Base de code prête pour ajouter de nouvelles mécaniques.
